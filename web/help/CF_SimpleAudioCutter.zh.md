基于 Qwen2-Audio-7B 的音频切片与 AI 分析节点。

- 音频文件：从已上传的音频中选择
- 关键帧JSON：时间戳（秒）的 JSON 数组，通过波形编辑器图形化管理
- 跳过初始段：开启=第一个标记仅作为起点（跳过之前的片段）
- 包含尾部段：开启=将最后一个标记到音频结尾也作为一段
- 分析指令：用自然语言告诉 AI 分析什么（如歌词、情感、乐器等）
- 分段索引：输出哪一段作为"当前分段音频"（1=第一段）
- 其他参数：控制 LLM 生成的温度、核采样、种子等

输出端口：
- 音频（合并）：所有选中片段合并
- 分段整数秒列表：每段的时长
- 分析文本：AI 分析结果
- 音频时长：总时长（秒）
- 当前分段音频：选中的单段
- 总段数
