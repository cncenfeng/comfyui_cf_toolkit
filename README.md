CF 工具包 (comfyui_cf_toolkit)
为 ComfyUI 设计的自定义节点包，提供图像管理、智能编辑、视频管理和音频切片功能，深度集成 Qwen2.5-VL 视觉语言模型。

1. 安装

环境要求: 最新版 ComfyUI，支持 Windows/Linux/macOS。

进入 ComfyUI 的 custom_nodes 目录：

```
cd ComfyUI/custom_nodes
```

克隆仓库：

```
git clone https://github.com/cncenfeng/comfyui_cf_toolkit
```

安装依赖：

```
cd comfyui_cf_toolkit
pip install -r requirements.txt
```

重启 ComfyUI。

2. 节点说明

所有节点位于 CF工具包 类别下，每个节点标题旁有 ? 按钮可查看详细帮助。

| 节点名称 | 功能描述 |
|----------|----------|
| CF 批量图像加载器 | 从指定子文件夹加载图像批次，提供固定高度、水平滚动的预览面板，支持上传、拖拽排序、删除、排除。可设置宽高缩放并统一输出尺寸。 |
| CF 批量图像管理器 | 接收图像批次，自动保存到输出目录，并重新读取输出完整批次。预览面板与加载器一致，支持删除和排序。覆盖模式可追加或覆写。 |
| CF 批量视频加载器 | 列出输入子目录下的视频文件，输出视频数量。支持 mp4/avi/mov/webm/mkv 格式。 |
| CF 批量视频管理器 | 管理输出目录下的视频文件，可选将全部视频拼接为一个文件（使用 ffmpeg）。覆盖模式控制拼接结果是否替换。 |
| CF 简易音频切片器 | 基于 Qwen2-Audio-7B 的音频切片与 AI 分析节点。通过关键帧时间标记进行切片，支持跳过初始段和包含尾部段。用自然语言输入分析指令，AI 自动分析音频内容。 |
| CF Qwen Image Edit (Batch Image) | 基于 Qwen2.5-VL 的智能图像编辑节点。使用 [pic:N] 标记动态选择主图和参考图，支持换装、多角色互动等复杂编辑。 |
| CF 通用运算器 | 集成算术、逻辑、比较、聚合共 20 余种运算模式。输入输出采用万能类型，自动返回整数、浮点数或布尔值。 |

3. 帮助系统

每个节点标题右侧有一个 ? 圆形按钮，点击可查看该节点详细的 Markdown 帮助文档。帮助文档自动匹配 ComfyUI 的当前语言设置：
- 英文界面显示 .en.md
- 中文界面显示 .zh.md

4. [pic:N] 标记语法详解

在 USER_PROMPT 中插入 [pic:N] 标记，节点将按提及顺序重新排列图片，第一个被提及的 [pic:N] 自动成为主图，输出到 first_image 端口。

示例：

图像批次: [Pic1: 人物], [Pic2: 服装A]

提示词: 让 [pic:1] 的人物穿上 [pic:2] 的衣服，背景改为纯白色。

效果: Pic1 为主图，参考 Pic2 服装生成最终图像。

支持多角色控制，例如：让 [pic:1] 穿上 [pic:6] 的衬衫，用 [pic:2] 的裤子，在 [pic:5] 的场景中。
